# YUC 66 部夏番目录 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 用 YUC 的 66 部夏番详细清单替代当前 63 部目录，同时维持已核验日本 TV 首播的北京时间周历。

**Architecture:** `data/anime.js` 仍是唯一静态数据源。YUC 负责目录成员、中文名、主视觉和官网，已有官方或 ORICON TV 时间继续由 `lib/schedule.js` 换算；不能确认的条目只进入目录区。

**Tech Stack:** Vinext、React 19、Node test、YUC 静态快照、动画官网与本地 PNG。
