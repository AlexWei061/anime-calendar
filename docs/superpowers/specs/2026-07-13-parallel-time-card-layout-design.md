# 并列时间卡片周历设计

## 目标

在七日周历中恢复“同一时间的作品并列”展示，同时保留本地封面，并让时间更容易扫读。

## 桌面布局

- 保留前一周、下一周、回到本周和周一至周日的七日列。
- 不再使用连续的 15:00 到次日时间轴，也不把节目按分钟向下顺排。
- 每个日列按原始 YUC 播出时间分组，组内节目依然并列。
- 每组左侧使用固定的加宽时间栏，显示如 20:30、次日 01:00；时间栏高度撑满同组的节目卡。
- 同组右侧的节目卡使用等宽网格，按数量并列；每张卡显示本地封面、中文名和集数。
- 组与组按原始播出时刻顺序垂直排列；相近但不相同的时刻不再互相覆盖。

## 时间与来源规则

- 作品继续留在 YUC 标注的播出日期列。
- 24:00、25:00、27:08 分别显示为次日 00:00、次日 01:00、次日 03:08。
- 展示转换不修改原始日期、首播日、集数或原始时间数据。

## 移动端与保留功能

- 移动端继续使用单日议程；同一时刻在窄屏下最多两张并列，更多卡片换行。
- 详情弹窗、网络放送卡片、来源链接和本地封面资源保持。

## 验证

- 覆盖同一时间节目被分到同一时间组且并列渲染。
- 覆盖次日显示时间仍在原日列。
- 覆盖定时节目卡使用本地封面，且页面没有旧的 lane 计算或连续 25 到 27 点轴。
