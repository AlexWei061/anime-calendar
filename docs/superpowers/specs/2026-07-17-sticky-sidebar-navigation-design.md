# 左侧导航吸顶设计

## 目标

桌面端向下浏览日历时，左侧“播出表”和“我的番剧”始终保持可见且可点击。

## 根因与方案

侧栏已使用 `position: sticky`，但作为网格子项会默认拉伸为整页高度，无法产生吸顶移动空间。

仅在 `.page-sidebar` 添加 `align-self: start`，使它按自身内容高度参与网格布局；现有 `position: sticky; top: 0` 随即生效。

## 边界与验证

- 不修改 JSX、导航状态或按钮语义。
- 小于等于 `860px` 时，现有 `position: static` 仍优先，移动端顶部导航不变。
- 添加 CSS 回归断言；验证桌面滚动时侧栏停留在视口顶部，并运行完整测试与项目范围 lint。
