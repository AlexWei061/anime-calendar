# 周历吸顶导航与表头设计

## 目标

在桌面端浏览一周时间轴的下半部分时，始终能看到周切换按钮和周一至周日的列标题；只有时间轴和节目卡随页面滚动。

## 范围

- 固定“上一周”“回到本周”“下一周”及当前周日期范围。
- 固定时间轴左上角空白单元格与七个星期表头，使其位于周导航下方。
- 固定效果只在桌面时间轴显示的断点生效；移动端单日议程保持现状。
- 固定区域仅在“一周放送安排”区间内有效，离开日历后自然解除，不能覆盖网络放送区。

## 实现

- 仅修改 `app/globals.css`，使用 CSS `position: sticky`、层级和背景色。
- 周导航贴住视口顶部；时间轴表头以周导航的高度为顶部偏移贴住其下方。
- 允许时间轴网格的吸顶表头溢出原网格边界，同时保留网格边框、背景和列对齐。
- 不调整 `app/page.tsx`，因此周切换、详情弹窗、数据计算和键盘行为都保持不变。

## 验证

- 以回归断言确认桌面 CSS 为周导航和表头设置了 `sticky`、正确层级与上下偏移，并且移动端不启用该效果。
- 运行现有完整测试与 lint。
- 在浏览器中滚动日历：检查导航和七个表头保持可见，节目卡在其下移动；继续滚过日历末尾，固定区域不遮盖网络放送内容。
